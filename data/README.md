# Required Local Data Files

The repository intentionally contains no dataset or annotation records.

Place the required files in a private/local Kaggle input dataset, or update the corresponding paths in `Config`:

- `train_tagged_ABSA_persian.csv`
- `test_tagged_ABSA_persian.csv`
- `ABSA_Dataset_train.csv`
- `ABSA_Dataset_food_test.csv`
- `ABSA_Dataset_movie_test.csv`
- `test_Pars_ABSA.csv`
- `human_only_test.csv`

These files are not redistributed here. Retain the original providers' licenses, consent conditions, privacy requirements, and attribution notices.
